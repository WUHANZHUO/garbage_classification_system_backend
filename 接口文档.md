

# 垃圾分类系统接口文档



## 用户模块

### **用户注册页面**

**功能描述：** 用户注册

**路径：**/api/auth/register

**类型：** POST

**请求参数：**

```json
{"username": "xx", "password": "yy"}
```

**参数描述：**

`username`（varchar，必需）：用户名

`password`（varchar，必需）：用户密码

**响应数据：**

①注册成功（201）：

```json
{ "message": "用户注册成功"}
```

②注册失败（409）：

```json
{"message": "用户名已存在"}
```

**备注：** 无



### 用户登录页面

**功能描述：** 用户/管理员登录

**路径：**/api/auth/login

**类型：** POST

**请求参数：**

```json
{"username": "xx", "password": "yy"}
```

**参数描述：**

`username`（varchar，必需）：用户名

`password`（varchar，必需）：用户密码

**响应数据：**

①登陆成功（200）：

```json
{

  "message": "登录成功",

  "role": 0,

  "token": "xx.yy.zz"

}
```

②登陆失败：

401:

```json
{"message": "用户名或密码错误"}
```

403 Forbidden：

```json
{'message': '该账户已被封禁，请联系管理员'}
```

**备注：**

role:0表示普通用户，1表示管理员

测试账号：

```json
{"username": "admintest", "password": "admintest"}

{"username": "usertest", "password": "usertest"}
```



### 修改个人密码 

**功能描述：**  登录用户修改自己的密码 

**路径：** /api/auth/change_password

**类型：**  PUT

**请求头：**

```html
Authorization: Bearer <用户token> 
Content-Type: application/json
```

**请求参数：**

```json
{
  "current_password": "old_password",
  "new_password": "new_password"
}
```

**参数描述：**

`current_password`（varchar，必需）：当前密码

`new_password`（varchar，必需）：新密码

**响应数据：**

成功响应 (200):

```json
{ "message": "密码修改成功" }
```

失败响应:

400 Bad Request:

```json
{"message": "请求参数不完整"}
```

401 Unauthorized:

```json
{"message": "缺少认证令牌"}
```

403 Forbidden:

```json
{"message": "当前密码不正确"}
```

```json
{"message": "该账户已被封禁"}
```

**备注：**  用户必须处于登录状态。

## 识别模块

### 文字识别

**功能描述：** 输入文字进行模糊匹配，返回垃圾类型，并获得积分。

**路径：** /api/recognize/text?q={string:content}

**类型：** GET

**路径参数：** q

请求示例：`http://ip:5000/api/recognize/text?q=香蕉`

**请求头：**

```html
Authorization: Bearer <用户token>

Content-Type: application/json
```

**响应数据：** 

成功响应：

```json
{
    "category": "有害垃圾",
    "id": 531,
    "name": "废电池"
}
```

```json
{
  "message": "未找到匹配的垃圾信息",
  "results": []
}
```

失败响应：

400 Bad Request:

```json
{"message": "缺少查询参数 q"}
```

404 Not Found:

```json
{"message": "未找到匹配的垃圾信息"}
```

**备注：** 无



### 图片识别

**功能描述：** 接收传入的图片，使用模型进行推理，输出识别的类型以及其置信度，并获得积分。

**路径：** /api/recognize/image

**类型：** POST

**请求头：** 

```html
Authorization: Bearer <用户token>

Content-Type: application/json
```

**请求参数：** 该接口的请求体类型为 multipart/form-data。

**参数描述：** `file`（File，必需）：用户上传的、需要进行垃圾分类识别的图片文件。

**响应数据：** 

成功响应：

```json
{
    "category": "clothes",
    "message": "识别成功",
    "probability": 0.8488699793815613
}
```

失败响应：

400 Bad Request：请求参数不正确。

```json
{
    "message": "缺少图片文件" 
}
```

401 Unauthorized：未提供或提供了无效的 token。

```json
{
    "message": "缺少认证信息"
}
```

422 Unprocessable Entity：上传的文件无法被模型正确处理。

```json
{
    "message": "图片文件无效或无法处理"
}
```

**备注：** 无
